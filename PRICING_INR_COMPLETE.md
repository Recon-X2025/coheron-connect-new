## Key Pricing Insights (INR)

### When to Use Modules vs. Plans

✅ **1-2 modules needed**: Standalone modules may be cheaper

✅ **5+ modules needed**: Plans are always better value

### Cost Savings Opportunities

1. **Annual Commitments**: Save 10-17% with annual billing
2. **Add-Ons**: Save 30-40% when adding modules to existing plans
3. **Bundles**: Save 20% when purchasing multiple modules together
4. **Plans**: Best value when you need 3+ modules
